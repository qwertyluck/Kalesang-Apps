# Kalesang Keuangan Satker

**Kalesang** (berasal dari bahasa Maluku yang berarti menjaga, merawat, dan memelihara) adalah aplikasi berbasis web personal yang dirancang untuk membantu tugas Bendahara Pengeluaran dalam melakukan pencatatan, pembukuan bayangan (*shadow BKU*), monitoring tagihan, manajemen aset, serta mitigasi risiko anggaran secara dinamis, terstruktur, dan akuntabel di satuan kerja.

---

## 🚀 Fitur Utama

* **Dashboard & Early Warning System:** Pemantauan real-time postur anggaran, sisa dana, dan pengingat tenggat waktu penting (pajak, batas UP/GU).
* **Shadow BKU (Pembukuan Bayangan):** Pencatatan transaksi harian mandiri untuk *drafting* dan validasi sebelum dieksekusi ke aplikasi resmi.
* **Monitoring Tagihan:** Pelacakan status dokumen pencairan anggaran dari tingkat SPP, SPM, hingga terbitnya SP2D.
* **Manajemen Aset & Persediaan:** Pencatatan detail barang belanjaan yang disesuaikan dengan referensi kode barang negara untuk mempermudah rekonsiliasi.
* **Mitigasi Risiko & Evaluasi:** Buku log khusus pencatatan temuan dan rencana aksi perbaikan untuk menjaga tertib administrasi keuangan.

---

## 🛠️ Teknologi yang Digunakan

* **Backend:** PHP / Node.js
* **Database:** MySQL / PostgreSQL
* **Frontend:** HTML, CSS (Bootstrap / Tailwind), JavaScript

---

## 📌 Status Proyek
Proyek ini dikembangkan secara bertahap sebagai sistem pendukung operasional personal (*shadow system*) untuk meningkatkan efisiensi dan akuntabilitas pengelolaan keuangan satuan kerja.
